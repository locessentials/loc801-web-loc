TEMA 3

HTML y Visual Studio Code

Desarrolla habilidades técnicas prácticas al aprender los fundamentos de HTML, seguir configurando tu entorno de desarrollo profesional y construir tu primer proyecto de página web bilingüe.

🎯 Objetivos de aprendizaje

  • Comprender la estructura de documentos HTML y elementos comunes usados en localización web
  • Identificar y trabajar con etiquetas HTML, atributos y contenido que requieren localización
  • Configurar Visual Studio Code con extensiones esenciales para trabajo de localización
  • Configurar GitHub Pages para hospedar tus proyectos de localización
  • Crear una página web bilingüe de terminología demostrando habilidades de localización HTML

📋 Trabajo que harás

Exposición
Código HTML

Aprende los fundamentos de HTML y cómo identificar elementos que necesitan localización en proyectos web.

Configuración
Configuración de VSC

Configura Visual Studio Code con extensiones que apoyan flujos de trabajo profesionales de localización.

Proyecto
Proyecto Página de Terminología

Construye tu primera página web bilingüe mostrando tus habilidades de localización en inglés y español.

⏱️ Tiempo estimado
Planifica aproximadamente 6-8 horas para completar todas las actividades de este tema. El Proyecto Terminology Webpage es tu primera asignación práctica de localización y se convertirá en parte de tu portafolio profesional.
